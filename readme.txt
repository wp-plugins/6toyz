Affiliation 6toyz

== Description ==

Affiliation 6toyz. Le plugin se connecte a la plateforme d'affiliation 6toyz.fr. En rentrant vos identifiants, vous allez pouvoir recuperer les flux d'informations de vos marques blanches.

== Installation ==

Extraire le fichier zip et deposez simplement le contenu dans le repertoire wp-content/plugins/ de votre installation de WordPress, puis activez le plugin depuis la page Plugins.
