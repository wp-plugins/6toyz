== Description ==

Affiliation 6toyz. Le plugin se connecte à la plateforme d'affiliation 6toyz.fr. En rentrant vos identifiants, vous allez pouvoir récupérer les flux d'informations de vos marques blanches.

== Installation ==

Extraire le fichier zip et d&amp;eacute;posez simplement le contenu dans le r&amp;eacute;pertoire wp-content/plugins/ de votre installation de WordPress, puis activez le plugin depuis la page Plugins.
